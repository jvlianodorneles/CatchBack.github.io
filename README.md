## Política de Privacidade – CatchBack

**Última atualização:** 6 de abril de 2026

A presente Política de Privacidade explica como o aplicativo **CatchBack** trata as informações dos usuários. Nosso compromisso é garantir a transparência e a segurança, cumprindo integralmente a **Política para Famílias do Google Play**, a Lei Geral de Proteção de Dados (**LGPD**) e a **COPPA**.

### 1. Compromisso com a Privacidade Infantil
O **CatchBack** foi projetado para ser um ambiente seguro para crianças e adultos. 
* **Sem Publicidade:** O aplicativo não exibe nenhum tipo de anúncio.
* **Sem Rastreamento Comportamental:** Não rastreamos o comportamento do usuário para fins de marketing ou criação de perfis.

### 2. Coleta e Uso de Dados
Para garantir a melhor experiência e funcionalidade do aplicativo, o tratamento de dados ocorre da seguinte forma:

* **Informações Pessoais:** O CatchBack **não coleta**, não solicita e não armazena informações de identificação pessoal (como nome, CPF, e-mail, endereço ou número de telefone).
* **Dados Técnicos e de Desempenho:** Podemos coletar dados técnicos anônimos (como versão do Android, modelo do aparelho e registros de erros) exclusivamente para melhorias de estabilidade e correções de bugs. Esses dados não são vinculados à identidade do usuário.
* **Permissões:** Caso o aplicativo solicite permissões específicas do sistema, elas são utilizadas apenas para o funcionamento das ferramentas do próprio app e nenhum dado resultante é transmitido a terceiros.

### 3. Compartilhamento de Dados
Não vendemos, alugamos ou compartilhamos qualquer dado de nossos usuários com terceiros. Como o aplicativo não utiliza SDKs de anúncios, não há compartilhamento de dados para fins publicitários.

### 4. Links para Terceiros
O aplicativo não contém links para sites ou serviços externos.

### 5. Retenção de Dados
Uma vez que não coletamos dados pessoais, não mantemos bancos de dados com informações privadas dos usuários. Caso você entre em contato via suporte, utilizaremos seu e-mail apenas para responder à sua dúvida, sendo a mensagem descartada após a resolução do problema.

### 6. Alterações nesta Política
Podemos atualizar esta política periodicamente. Recomendamos que os pais ou responsáveis revisem esta página para se manterem informados. O uso continuado do aplicativo após alterações constitui a aceitação dos novos termos.
